# autism_prediction
A quick, interactive web-based screening quiz for Autism Spectrum traits, powered by a machine learning model (Random Forest) trained on the Autism Screening Adult dataset.  ⚠️ Disclaimer: This is an informational screening tool, not a medical diagnostic instrument. Results should never replace evaluation by a qualified healthcare professional
